## 📋 Features — Detailed Explanations

### 1. Rechnungserfassung per WhatsApp
- **What:** Client sends a photo of an invoice via WhatsApp
- **How:** AI extracts: vendor, amount, date, tax rates, invoice number
- **Result:** Data is automatically entered into DATEV/lexoffice/sevDesk
- **Accuracy:** 98%+ recognition rate
- **Time saved:** ~2 hours/day per Kanzlei
- **Example workflow:** Foto senden → KI erfasst → Prüfen → DATEV-Export

### 2. Automatische Bankabstimmung
- **What:** Send bank statements via WhatsApp
- **How:** AI matches transactions against open items, flags discrepancies
- **Result:** Complete reconciliation in minutes instead of hours
- **Key value:** Detects double bookings, missing entries, anomalies
- **Time saved:** ~18 hours/week for a 140-client Kanzlei

### 3. USt-VA vorbereiten
- **What:** AI prepares Umsatzsteuervoranmeldung data
- **How:** Aggregates all VAT-relevant transactions from the month
- **Result:** Ready-to-file VAT return — just review and submit
- **For:** Professional plan only

### 4. DATEV-Export
- **What:** Seamless export to German tax software
- **Supported:** DATEV, lexoffice, sevDesk, and more
- **How:** One-click export from WhatsApp, no manual re-entry

### 5. Mandanten-Kommunikation
- **What:** Clients can message the Kanzlei via WhatsApp
- **How:** @AsTech answers standard questions automatically, routes complex ones to staff
- **Benefit:** 24/7 availability, instant responses, reduced phone calls

### 6. Fristen-Überwachung
- **What:** Automatic deadline monitoring
- **Tracks:** Steuertermine, Abgabefristen, Zahlungsziele
- **Alerts:** Push notifications via WhatsApp before deadlines
- **For:** Professional plan only
